# dotnet6Test

This project is a demo project with dotnet 6. Also include some good practices such as batect, truffhog, lint when you are developing a new project.  

## Pre-Requisite  

* Dotnet 6 SDK
* JDK for batect
* Docker


### Batect Tasks

```bash
./batect --list-tasks
```

### Run App

```bash
./batect run
```
